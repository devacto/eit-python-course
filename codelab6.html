Code Lab 6

Hi and welcome to code lab 6! In this code lab, we are going to start interacting with the user by asking them some questions.

Exercise 1 - Asking questions!

In this first exercise, we are going to try to get the computer to ask us some questions and print the answers onto the screen.

# This program asks some questions and will display the user's answer
# Victor Wibisono, 2013

print "How old are you?",
age = raw_input()
print "How tall are you?",
height = raw_input()
print "How much do you weigh?",
weight = raw_input()

age1 = raw_input("How old are you?")
height1 = raw_input("How tall are you?")
weight1 = raw_input("How much do you weigh?")
print "So, you are %r years old, %r tall and %r heavy." % (age, height, weight)

Now run the code in the terminal by pressing the "Run" button and see the result.

Remember to write down your mistakes if you have made any, and try to reduce them as we go along these exercises.

Go Further

Congratulations for having completed the exercise. Now it's time to go further by learning about 'pydoc'.

  1. Go online and find out what Python's raw_input() does.
  2. In the terminal where you normally run python to run your scripts, type "pydoc raw_input" (without the "). Read what it says.
  3. What does the 'pydoc' command do?
  4. Use pydoc to also read about 'open', 'file', 'os', and 'sys'. It's alright if you do not understand those. Just read through and take notes about interesting things.
