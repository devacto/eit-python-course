Exercise 1 - Parameters, Unpacking, Variables

In this first exercise, we are trying to pass variables to a Python script.

# This program demonstrates passing variables to a Python script
# Victor Wibisono, May 2013

from sys import argv

script, first, second, third = argv

print "The script is called:", script
print "Your first variable is:", first
print "Your second variable is:", second
print "Your third variable is:", third

Now run the code in the terminal by pressing the "Run" button and see the result. Find out what a Python 'module' is. How do you insert variables into a Python script in Eclipse?

Exercise 2 - Prompting and Passing

In this exercise, use argv and raw_input together to ask the user something specific.

# This program demonstrates the use of argv and raw_input
# Victor Wibisono, May 2013

from sys import argv

script, user_name = argv

prompt = '> '

print "Hi %s, I'm the %s script." % (user_name, script)
print "I'd like to ask you a few questions."
print "Do you like me %s?" % user_name
likes = raw_input(prompt)

print "Where do you live %s?" %s user_name
lives = raw_input(prompt)

print "What kind of computer do you have?"
computer = raw_input(prompt)

print """
Alright, so you said %r about liking me.
You live in %r. Not sure where that is.
And you have a %r computer. Nice.
""" % (likes, lives, computer)

Now run your code by pressing the "Run" button. Don't forget to put in the necessary parameters. Let us know if you face any issues!
