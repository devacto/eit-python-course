Code Lab 4

Welcome to the second day of the course! Hope you are enjoying the course so far. In this code lab, we are going to dive deeper into variables and printing.

Exercise 1 - Variables and printing

In this first exercise, we are going to try to get the computer to print something onto the screen.

# A program that demonstrate variables and printing
# Victor Wibisono, May 2013

my_name = 'Victor Wibisono'
my_hair = 'Black'
my_programming_language = 'Python'
my_age = 22
my_height = 165 # centimeters
my_weight = 80 # kg

print "Hello, my name is %s." % my_name
print "My favourite programming language is %s." % my_programming_language
print "I am %s years old." % my_age

Now run the code in the terminal by pressing the "Run" button and see the result. Find out why the program is throwing an error.

Exercise 2 - More printing of variables

Good work on having written the program to print strings into the screen. Now what about some numbers! Add the following codes into your codelab4.py file.

print "I am %d years old." % my_age
print "I am %d cm tall and weigh %d kg." % (my_height, my_weight)
print "If I add %d, %d, and %d, I get %d." %
    (my_age, my_height, my_weight, my_age + my_height + my_weight)

Now run your code again by pressing the "Run" button.
