Welcome to the EIT's Python Course

Welcome to EIT's introductory programming course. This course will be done in Python.

Why learn a programming language?

Why Python?

  • Python is widely used. Python is used widely in Google and on its products, and it is widely used in the Internet security circle.
  • It is very easy to get started in Python.
  • Python codes are very readable.
  • Python codes are very easy to understand because they are modular and are abstracted.

Course Organisation

This course will run from Monday, 24 June to Friday, 28 June 2013.

Sessions

Each day, there will be four CodeLab sessions. So in total, there will be 20 CodeLab sessions.

  • Session 2: 1100 - 1230
  • Session 3: 1400 - 1530
  • Session 4: 1600 - 1730

Each session will consist of the following session components:

  • Exercise 1
  • Exercise 2
  • Session Review
