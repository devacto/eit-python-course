Code Lab 2

Great work on completing the first two code lab sessions! Now you know how to print stuff on the screen. Now we will learn how to get the computer to do simple mathematics.

Exercise 1 - Numbers and math

For the first exercise, we will get the computer to do some simple mathematics for us.

# A Simple Math Application
# Victor Wibisono, May 2013

print "Let me count my chickens."
print "Hens", 2 + 3
print "Roosters", 100 - 25 * 3 % 4

print "Is it true that 3 + 2 < 5 - 7 ?"
print "3 + 2 = ", 3 + 2
print "5 - 7 = ", 5 - 7

Exercise 2 - More math

Good job on getting through the previous exercise. Now what about some more complex mathematics?

# A Slightly More Complex Math Application
# Author: Victor Wibisono (devacto@gmail.com)

print "10 / 3 = ", 10 / 3
print "10 % 3 = ", 10 % 3

Go Further

Congratulations for having completed the two exercises. Now it's time to go further and learn more by answering these questions:

  1. What is the difference between the '/' operator and the '%' operator?
  2. Notice that the math seems "wrong"? There are no fractions, only whole numbers. Find out why by researching what a "floating point" number is.
  3. Rewrite exercise 2 using "floating point" number so that it is more accurate.
