Exercise 1 - Variables and names

Now you can print things with print and can do math. The next step is to learn about variables. In programming, a variable is nothing more than a name we give to something we program. This is useful so we will refer to the more meaningful name rather than its less meaningful value.

# This program demonstrate the use of variables.
# Author: Victor Wibisono, May 2013

cars = 100
space_in_a_car = 4.0
drivers = 30
passengers = 90
cars_driven = drivers

print "The number of cars in the garage is ", cars
print "There are ", space_in_a_car, " spaces available in a car."
print "There are only ", drivers, " drivers available."

Exercise 2 - More variables! And maths!

Now add the following codes to the same codelab3.py file.

cars_not_driven = cars - drivers
carpool_capacity = cars_driven * space_in_a_car
average_passengers_per_car = passengers / car_driven

print "So the number of cars that are not driven is ", cars_not_driven
