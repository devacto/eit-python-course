Exercise 1 - Reading files

In this first exercise, we are going to write a program that reads an input from a text file. This file involves two files - one is a .txt file and another one a .py file. First off, make a .txt file with the following content.

This is a text file,
that should be read by a Python script.

Now in the .py file, write the following.

# This program reads a file
# Victor Wibisono, May 2013

from sys import argv

script, filename = argv

txt = open(filename)

print "Here is your file %r:" % filename
print txt.read()

print "I'll also ask you to type it again:"
file_again = raw_input("> ")

txt_again = open(file_again)

print txt_again.read()

Go Further

Congratulations for having completed the exercise. Since this is quite a complicated concept, we are only doing one set of exercise. Now it's time to go further and learn more by answering these questions:

  1. Above each line, write comments in English saying what the line does. If you are unsure, inform an instructor.
  2. Search around online to see what other people do to define "functions" and "methods".
  3. Get rid of the part from line 10 - 15 where you use raw_input and try the script then.
  4. Run "pydoc file" on the terminal and scroll down until you see the read() command (method / function). See all the other ones you can use? Skip the ones that have __ (two underscores) in front because those are irrelevant. Try some of the other commands.
  5. Have your script also do a close() on the txt and txt_again variables. It's important to close file when you are done with them.
